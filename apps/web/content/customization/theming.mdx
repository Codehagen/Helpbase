# Theming

Your help center uses CSS variables for theming, making it easy to customize colors, fonts, and spacing.

## Color system

Colors are defined using the oklch color space for perceptually uniform color manipulation:

```css
:root {
  --background: oklch(1 0 0);
  --foreground: oklch(0.153 0.006 107.1);
  --primary: oklch(0.228 0.013 107.4);
  --muted: oklch(0.966 0.005 106.5);
}
```

## Dark mode

Dark mode is built-in and works automatically. Users can toggle it with the `d` key or the theme toggle in the header.

The `.dark` class switches CSS variables to dark values:

```css
.dark {
  --background: oklch(0.153 0.006 107.1);
  --foreground: oklch(0.988 0.003 106.5);
}
```

## Typography

The help center uses [Geist](https://vercel.com/font) — a clean, modern font family by Vercel. Both sans-serif and monospace variants are included.
